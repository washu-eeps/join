# Join the WashU EEPS GitHub Organization

Welcome! This repository is used to request membership in the [washu-eeps](https://github.com/washu-eeps) GitHub organization.

## How to Request Access

1. [Open a new issue](https://github.com/washu-eeps/join/issues/new) in this repository
2. Include the following information:
   - Your **full name**
   - Your **WashU email address**
   - Your **GitHub username**
   - Your **role** (faculty, graduate student, undergraduate, staff)
   - The **course or project** you need access for (e.g., EDC 2026, EEPS 3230)
3. An administrator will add you to the appropriate team and close the issue

## Teams

| Team | Description |
|------|-------------|
| `students` | Graduate and undergraduate members |
| `courses` | Instructors and TAs |
| `projects` | Research collaborations |
| `admins` | Organization administrators |
| `faculty` | Faculty members |

## Questions?

Contact the org administrators or your course instructor.
